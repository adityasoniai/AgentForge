# Contributing to AgentForge

Thanks for considering a contribution! This repo is meant to grow with the community.

## Ways to Contribute

- **Fill in a stub section** — most Parts currently have a finalized structure but need full write-ups, diagrams, and code examples (Part I is the reference quality bar).
- **Fix errors** — technical inaccuracies, outdated tool information, broken links.
- **Add examples** — real, runnable code examples are especially valuable.
- **Update the "What Changed This Month" log** — the GenAI/agent space moves fast; help keep the repo current.
- **Suggest new capstone projects or interview questions.**

## Guidelines

1. Match the existing format: concept → diagram (where useful) → example → checkpoint/mini-project.
2. Keep tool comparisons in the central [Appendix](appendix/README.md) rather than repeating tables in every section.
3. Cite sources for any claims about specific benchmarks, pricing, or release dates — the ecosystem changes monthly, so unsourced claims age badly.
4. Use Mermaid syntax for diagrams where possible (renders natively on GitHub, no external image hosting needed).
5. Open an issue before starting large sections, so effort isn't duplicated.

## Style

- Beginner-friendly language, even in advanced sections — define jargon on first use or link to the [Glossary](appendix/README.md#a1-glossary-of-terms).
- Difficulty tags (🟢🟡🔴) should reflect actual prerequisite knowledge, not just section order.

We're excited to build this with you. 🚀
